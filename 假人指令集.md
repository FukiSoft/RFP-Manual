# 假人指令合集
> [!TIP]
> 您可以使用 Ctrl+F 组合键进行搜索；

## 生存服三界伪和平
- 末地伪和平：
```
/player end_whp spawn at 4082.5 35.00 -3623.5 facing 90 45 in minecraft:the_end 
/player end_whp use
```
- 下界伪和平：
```
/player nether_whp spawn at 1447.5 128.00 5213.50 facing 0 45 in minecraft:the_nether 
/player nether_whp use
```
- 主世界伪和平：
```
/player whp spawn at 1434.5 129.00 5210.5 facing 0 30 in minecraft:the_nether 
/player whp use
```

> [!TIP]
> 三界伪和平均为堆叠2000只凋零弱加载版本，一般重启服务器后除下界伪和平外都只需要放出假人即可重新启用伪和平，
> 仅下界伪和平需要手动use两次重启伪和平；

## 猪人塔
> [!WARNING]
> 猪人塔必须先挂假人开砍使砍怪架开机再关闭伪和平，否则猪人会堆积在砍怪架！

```
/player zrt1 spawn at 1152 34 2255 facing 0 45 in minecraft:the_nether
/player bot_zrt attack interval 10
/player bot_nether_whp use
```
